# Contributing to VaultNote

Thanks for your interest in contributing! Please follow these steps:

1. Fork the repo and create a feature branch.
2. Keep changes focused and add tests where applicable.
3. Run `npm ci` and `npm run build` locally.
4. Open a pull request with a clear description and link to relevant issues.
5. Follow the Code of Conduct.

We use PRs for all code changes. CI must pass before merging.
